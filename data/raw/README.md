# Raw data

Set `UCFCRIME2LOCAL_ROOT` to your UCF-Crime2Local directory (contains `rgb-images/`, `labels/`, `train.txt`, `test.txt`, `labels.txt`).

Example:

```bash
export UCFCRIME2LOCAL_ROOT=/path/to/ucfcrime2local
```

You can symlink the dataset here as `data/raw/ucfcrime2local` if you prefer a fixed path.
